## 🧪 Example Console Invocation

Launch Outwrite AI from your terminal with a single command:

```bash
# Basic launch — loads default profile
outwrite --config ./outwrite.config.json

# Launch with a specific document
outwrite --load ~/Documents/report_v1.md

# Enable offline mode explicitly
outwrite --offline --config my_custom_profile.json

# Launch embedded in a browser window (headless mode)
outwrite --headless --port 8080 --no-gui
```

> 💬 **Console output example:**
> ```
> Outwrite AI v2026.03.21
> Profile: Professional Writer
> Mode: Offline
> Listening on http://localhost:8080
> API Status: OpenAI (connected), Claude (standby), Local (active)
> ```

## 🔗 OpenAI & Claude API Integration

Outwrite AI natively supports both **OpenAI GPT-4** and **Anthropic Claude** models. You can switch between them dynamically—or chain them for multi-model reasoning.

| Feature | OpenAI | Claude |
|---------|--------|--------|
| Real-time suggestions | ✅ | ✅ |
| Long document processing | ✅ | ✅ (up to 100k tokens) |
| Code generation | ✅ | ✅ |
| Sentiment analysis | ✅ | ✅ |
| Cost optimization | ✅ (token budgets) | ✅ (token budgets) |

To enable:

1. Obtain API keys from [OpenAI](https://platform.openai.com) or [Anthropic](https://console.anthropic.com).
2. Add them to your config (see example above).
3. Restart Outwrite AI. Both engines will appear in the model selector.

> 🛡️ *Your keys never leave your machine. All requests route directly from Outwrite AI to the API endpoint.*
